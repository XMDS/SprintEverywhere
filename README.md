# SprintEverywhere

## English
 A plugin for GTASA for Android, you can run and sprint quickly anywhere, such as indoors, you can also enjoy faster running speed.

 ## 中文
 一个GTASA安卓版插件，让你在游戏的任何地方都能快速奔跑，比如在室内，你也可以奔跑。
